# **App Name**: Na-NEET Prep

## Core Features:

- Curated NEET Content: Access high-quality, hand-crafted study notes for Physics, Chemistry, and Biology.
- PYQ Access: Browse NEET Previous Year Questions (PYQs) organized year-wise and chapter-wise with original explanations for focused preparation.
- In-App YouTube Player: Stream educational videos directly from the admin's YouTube channel without redirection.
- Workspace / Mind Maps: View mind maps created and managed by the admin to aid visual learning and information retention.
- Dynamic Content Updates: Study materials are dynamically updated from the backend, ensuring content is always current.
- Restricted Admin System: Secure admin login accessible only via the verified Gmail ID (naveenatariya999@gmail.com), allowing content management, including upload, edit, delete and restore functionalities.
- Admin Google Drive Integration: Upload and manage PDF files and study content securely to Google Drive via the Google Drive API.
- Admin Filter Control System: Admin can use filter controls to show/hide or enable/disable content by subject, content type or entire section, without permanently deleting it.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke a sense of focus and intellect.
- Background color: Light Grey (#F0F2F5), providing a distraction-free environment.
- Accent color: Amber (#FFC107) for highlighting key information and interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif with a modern look, for optimal readability.
- Code font: 'Source Code Pro' for any code snippets within study material.
- Clean and minimalistic layout, optimizing screen space for study content and reducing distractions.
- Subtle transitions and animations for smooth navigation and content loading.
